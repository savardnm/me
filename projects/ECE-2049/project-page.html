Embedded Systems Programming

Project Description

In embedded systems, we worked with a low-power microprocessor, the MSP430 to develop small programs in C. These programs used a handful of features of the chip including: Digital I/O, Timers, Interrupt Service Routines (ISRs), and ADC Analog Input. These features relied on modifying the registers of the board to change pin modes, configure timers, and activate functions.
